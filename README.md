# my-project-tracker

v12 實裝「詳細描述」與「留言系統」
我覺得 第 1 點 (描述與留言) 是目前最缺少的靈魂功能。加入了這兩樣，它才算是一個完整的 Project Management Tool。

這是 Project Manager v12 (Collaboration Edition)。

✨ v12 更新重點：
📝 詳細描述 (Description): 任務詳情增加大方格，讓你寫 Spec 或備註。

💬 留言系統 (Comments): 可以在任務內「發送留言」，系統會自動記錄是誰(Assignee)在幾點發的言。

⌨️ 快捷鍵支援: 加入了 N (新增) 和 Esc (關閉) 的快捷鍵。

🌟 v13 新增功能說明：
📅 日曆視圖 (Calendar View):

在導航欄切換「📋 看板 / 📅 日曆」。

自動將有「截止日期」的任務顯示在日曆格子上。

支援月份切換 (上個月/下個月)。

📎 檔案附件 (Attachments):

在任務詳情中，你可以點擊「上傳檔案」（模擬）或「添加連結」。

如果是上傳，系統會讀取檔名並顯示在列表（不佔用大量內存，防止崩潰），看起來就像真的附件列表。

🚨 優先級排序 (Auto Sort):

新增「🚨 按優先級排序」按鈕。一按下去，看板內的所有任務會自動重新排列：High 在最上，Low 在最下。

👀 專注模式 (Focus Mode):

新增「👀 專注模式」按鈕。

開啟後，Blocked 和 Done 欄位會被隱藏，讓介面變寬，讓你專心處理未完成的工作。
